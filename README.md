# PalindromeCheckerApp
## Running Procedure
1. **Compile:** `javac UseCase1PalindromeCheckerApp.java`
2. **Run:** `java UseCase1PalindromeCheckerApp`

## Flow of Project
1. Program starts execution.
2. JVM invokes the `main()` method.
3. Application name and Welcome Message are displayed.
4. Application Version is shown.
5. Program completes startup and exits (awaiting next phase).

## Objective
To establish the entry point of the application, ensuring the environment is set up correctly and the user is greeted with the correct application details before any logic is processed.

## What topics will be covered
* **Class Structure:** Creating the main container.
* **Main Method:** Understanding `public static void main(String[] args)`.
* **Static Keyword:** Invoking methods without objects.
* **Console Output:** Using `System.out.println`.
* **Flow Control:** Basic sequential execution.
